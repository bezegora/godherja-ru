### Table List

# hunting_game_effect - unifies universal information about game (name, tier, prowess) into a single spreadsheet.
## Valid MODE:
### get_tags - If hunt_target_game is set, stores all of the creatures tags as boolean values
### combat - if hunt_target_game is set, executes a simple duel based on the creature's prowess

# (game_name)_game_tags_effect - sets a creature's tags.
## Valid MODE:
### set - If hunt_target_game is set, stores all of the creatures tags as boolean values. Only used internally
### get_reward - Applies the rewards for killing the animal to root
### get_allowed - Copies all game that a hunter can pursue into a temporary allowed_game list
### increase_proficiency - Increases a hunter's proficiency against the current game
### set_expert - Sets a hunter's proficiency to expert against the current game
### downgrade_proficiencies - Downgrades all major proficiencies to medium

## Valid tags (at the time of writing):
### skittish, cautious, aggressive, unyielding, cunning, stupid, fast, slow, large, elusive

# biozone_effect - unifies universal information about biozones (name, ambient danger, geographic region) into a single spreadsheet
